# 화평법(K-REACH) 규제물질 리스트 다운로드 방법

> **대상 사이트:** [화학물질정보처리시스템 (KREACH)](https://kreach.mcee.go.kr/repwrt/index.do)

---

## 절차

### 1단계 — 사이트 접속

아래 URL로 접속합니다.

```
https://kreach.mcee.go.kr/repwrt/index.do
```

---

### 2단계 — 화학물질 통합검색 실행

메인 화면에서 **「화학물질 통합검색」** 입력 틀 안의 **\[검색\]** 버튼을 클릭합니다.

> 별도의 검색어 없이 검색 버튼만 클릭하면 전체 물질 목록이 조회됩니다.

---

### 3단계 — 엑셀 다운로드

검색 결과 화면에서 **\[엑셀 다운로드\]** 버튼을 클릭합니다.

> 물질 수에 따라 다운로드에 수십 초가 소요될 수 있습니다.

---

### 4단계 — 다운로드 파일 열기

다운로드가 완료되면 아래 경로에서 파일을 확인합니다.

```
C:\Users\PC-86\Downloads\
```

파일명 형식:

```
화학물질_YYYYMMDD.xlsx
```

예시 (오늘 날짜 기준):

```
화학물질_20260608.xlsx
```

> **주의:** 파일명 뒤의 날짜(YYYYMMDD)는 다운로드 실행 당일 날짜로 자동 생성됩니다.  
> 오늘 날짜에 해당하는 파일을 열면 됩니다.

---

### 5단계 — 결과물 저장

다운로드된 파일을 아래 경로에 날짜별 폴더를 만들어 저장합니다.

**저장 기본 경로:**
```
outputs\화평법_규제DB_결과물\YYYYMMDD\
```

폴더 구조 예시:
```
outputs\
└── 화평법_규제DB_결과물\
    ├── 20260608\
    │   ├── 화학물질_20260608.xlsx       ← 원본 다운로드 파일
    │   └── 업데이트_보고서_20260608.md  ← 검증 및 통계 보고서
    └── 20260701\
        ├── 화학물질_20260701.xlsx
        └── 업데이트_보고서_20260701.md
```

> 같은 날짜에 여러 번 실행하면 기존 파일을 덮어씁니다.
