# 화평법(K-REACH) 화학물질 규제대응 시스템

화학물질의 등록 및 평가 등에 관한 법률(화평법, K-REACH) 규제 대응을 위한 MD/Skill 모음입니다.

## 폴더 구조

```
.
├── docs/                          # 운영 문서
│   ├── 다운로드_방법.md            # KREACH 사이트 엑셀 다운로드 절차
│   └── 엑셀_구조_가이드.md         # 화학물질 엑셀 구조 및 변환 규칙 (AI 명령서)
│
├── skills/
│   └── kreach-db/                 # K-REACH DB 업데이트 Cowork Skill
│       ├── SKILL.md               # 스킬 메인 (트리거 + 실행 순서)
│       ├── scripts/
│       │   └── transform.py       # 13열→20열 변환 스크립트 (K열 파싱)
│       └── references/
│           ├── 다운로드_방법.md
│           └── 엑셀_구조_가이드.md
│
└── results/                       # 업데이트 실행 결과 보고서
    └── 20260608/
        └── 업데이트_보고서_20260608.md
```

## 주요 기능

| 기능 | 설명 |
|------|------|
| DB 다운로드 | kreach.mcee.go.kr에서 화학물질 전체 목록 엑셀 다운로드 |
| 구조 검증 | 13열(A~M) 헤더 자동 검증 |
| 컬럼 변환 | K열 텍스트 파싱 → N~T열 수치 추출 (47,517건) |
| 결과 보고 | 규제물질 현황 통계 자동 생성 |

## 변환 컬럼 (N~T열)

| 열 | 내용 | 출처 |
|----|------|------|
| N | 인체급성유해성 (%) | K열 파싱 |
| O | 인체만성유해성 (%) | K열 파싱 |
| P | 생태유해성 (%) | K열 파싱 |
| Q | 제한물질 (%) | K열 파싱 |
| R | 금지물질 (%) | K열 파싱 |
| S | 허가물질 (%) | K열 파싱 |
| T | 사고대비물질 번호 | G열 복사 |

## 사용 방법 (Claude Cowork)

1. `kreach-db.skill` 파일을 Cowork에 설치
2. "화평법 규제물질 리스트 업데이트 해줘" 요청
3. 자동으로 다운로드 → 검증 → 변환 → 저장 실행

## 관련 법령

- 화학물질의 등록 및 평가 등에 관한 법률 (화평법, K-REACH)
- 소관: 환경부
- 시스템: [화학물질정보처리시스템](https://kreach.mcee.go.kr)
